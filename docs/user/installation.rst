Installation
============

Pip & Crate.io
--------------

Installing Jeeves is simple with `pip <http://www.pip-installer.org/>`_::

    $ pip install jeeves-framework

or, if pypi is down using `crate.io <http://www.crate.io>`_::

    $ pip install -i https://simple.crate.io/ jeeves-framework
